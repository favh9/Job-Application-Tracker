# Job-Application-Tracker

## MVP - A simplistic interface to store and organize job applications

### Necessary
- A quick way to add/update/remove/view a job application

A view like: </br>
```Company Name``` ```Website Link``` ```Job Title``` ```Status```

And then a sidebar or a new page that opens when I select the company, showing a quick way to add/view/update/remove the following: </br>
- Resume and/or (download link options .docx .pdf)
- When it was created/submitted
- Interview Date
- Notes (research, pros/cons, skills to demonstrate)

---
### Extras
- Visualize the status overall of my applications
- AI to see validate my resume against the job

## **_Getting started_**
Create a virtual environment to create an isolated directory for the project's dependecies.

### 1. Create a Virtual Environment:
- ```python3 -m venv .venv```

### 2. Activate the Environment:

- macOS/Linux:
    - ```source .venv/bin/activate```

- Windows (Command Prompt):
    - ```.venv\Scripts\activate```

### 3. Install Requirements
- ```python3 -m pip install -r requirements```

## Exiting the Environment
Enter the following command: ```deactivate```
